# Discord-Image-Classification-Bott 

My project can be used to detect some sports balls. You just need to send a photo and the programm will detect and also write a procent what she thinks it could possibly be. There are about 9 tyoes of different balls. The whole programm is in the Telegram. So it it`s a TelegramBot. Later on the picture you can also see some squars which a programm do to show you where the object is on the picture. In the future we will add some more settings to customize it.
